\noindent
This is the running record of the episcaf project: scaffolding antibody epitopes defined by
3D geometry into folded, assay-ready designs (RFdiffusion3 $\rightarrow$ ProteinMPNN
$\rightarrow$ AlphaFold3). It is meant to be read in order, since each section is one
iteration: what we tried, what we found, and what it changed for the next step. It is not a
formal manuscript. Every figure is regenerated by a named script (recorded in
\texttt{manuscript/figures/FIGURES.md}) and every number traces to \texttt{results/};
unconfirmed values are flagged \texttt{[verify]}.
